About "Travel Planner"

This application demonstrates a full-stack approach using the following technologies:

  • Backend: Flask (Python)
  • Frontend: React (loaded via CDN with JSX)
  • Database: SQLAlchemy (with SQLite)
  • Authentication: Flask-Login (cookie-based sessions)
  • API: Custom REST API, OpenWeatherMap (weather), Nominatim (search)
  • Maps: Leaflet.js
  • Styling: Tailwind CSS
